# 방식

- 매주 2주제를 정한다.
- 해당 주제의 출제자를 정한다.
- 한 세션에서 모두가 같은 주제를 다룬다.
- 각자 2주제에 대해 열심히 공부한다.
- 수 21시, 금 9시 매주 2회 진행한다.

        - 본 스터디는 주 2회 진행하므로, 깊에 공부하기에 무리가 있을 것 같아 거부되었다.
